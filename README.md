# SLM
## Skahd's Lootplot Mod, for lootplot!

## DOWNLOAD (NEW)
1. download entire zip through code -> download zip
2. unzip it
3. go inside the folder
4. select the slm's mods you want
5. move them to %appdata%/lootplot/mods
